# DATASET.md

## 1. 数据来源

当前原型包含 4 类数据：

1. **运营手动输入**：产品名、卖点、用户、活动信息。
2. **飞书营销表模拟数据**：品类表现、历史评论、私聊、成交、素材记录。
3. **素材库数据**：飞书素材表、本地素材库、图片搜索 API 模拟结果。
4. **运营反馈数据**：曝光、点赞、评论、私聊、成交、潜在客户沟通摘要。

## 2. 产品输入字段

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 产品名称 | 文本 | 是 | 需要生成文案的产品 |
| 产品类型 | 枚举 | 是 | 鼠标、键盘、耳机、电竞椅等 |
| 核心卖点 | 多行文本 | 是 | 轻量、低延迟、灯效、舒适等 |
| 目标用户 | 文本 | 是 | FPS 玩家、学生党、宿舍党等 |
| 活动信息 | 多行文本 | 否 | 价格、赠品、库存、活动时间 |
| 发布目的 | 枚举 | 是 | 种草、促销、上新、清库存等 |
| 想要语气 | 枚举 | 否 | 玩家口吻、朋友安利、轻松吐槽等 |
| 禁用词 | 文本 | 否 | 需要规避的风险词或广告词 |

## 3. 飞书连接字段

| 字段 | 说明 |
|---|---|
| App Token | 飞书应用或多维表格 token |
| Table / Sheet ID | 多维表格 table id 或电子表格 sheet id |
| View ID | 可选，用于限定视图 |

当前原型不会真的请求飞书，只读取这些字段并模拟返回。真实接入时使用飞书开放平台接口。

## 4. 素材字段

| 字段 | 说明 |
|---|---|
| 素材来源 | 飞书素材表、本地素材库、图片搜索 API |
| 配图关键词 | 黑色鼠标、桌搭、开黑、灯效等 |
| 排版样式 | 朋友圈单图海报、三图卡片、九宫格首图 |

素材记录结构：

```json
{
  "id": "fs-001",
  "type": "无线鼠标",
  "title": "黑色鼠标桌搭实拍",
  "tags": ["无线鼠标", "黑色", "桌搭", "FPS"],
  "source": "飞书素材表"
}
```

## 5. 运营反馈字段

| 字段 | 用途 |
|---|---|
| 曝光 | 计算互动率、私聊率、成交率 |
| 点赞 | 判断内容吸引力 |
| 评论 | 判断话题性和潜在问题 |
| 私聊 | 判断购买意向 |
| 成交 | 判断转化效果 |
| 近期热卖品类 | 调整内容重点 |
| 潜在客户沟通摘要 | 提炼用户疑虑和下一条文案方向 |
