# ASSET_PIPELINE.md

## 1. 目标

配图不再只是“建议”，而是由 Agent 自动检索、选择并排版。

## 2. 素材来源

1. 飞书素材表：运营上传的产品图、桌搭图、细节图。
2. 本地素材库：品牌已有素材文件夹。
3. 图片搜索 API：用于补充场景灵感，生产环境需注意版权。

## 3. 自动找图逻辑

输入：

- 产品类型
- 核心卖点
- 配图关键词
- 潜在客户沟通摘要
- 素材来源

打分：

| 规则 | 分数 |
|---|---|
| 素材品类等于产品类型 | +8 |
| 标签命中关键词 | +2 |
| 潜客提到实拍/细节/桌面，素材命中 | +4 |

输出：

- 首图
- 补充图
- 匹配分
- 素材来源

## 4. 自动排版逻辑

支持：

- 朋友圈单图海报
- 三图种草卡片
- 九宫格首图

排版结构：

```text
首图素材
产品名称
3 个核心卖点
文案摘要
轻 CTA
```

## 5. 当前原型实现

当前用 HTML/CSS 生成视觉预览，便于面试演示。

真实上线可升级：

- Canvas 导出 PNG
- Figma API 生成模板图
- Canva API 生成模板图
- 品牌素材 CDN/OSS 存储
