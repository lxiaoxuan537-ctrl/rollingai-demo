# 电竞朋友圈种草运营 AI Agent 小应用

这是一个面试可演示的轻量 AI Agent 原型，用于把电竞产品信息、飞书营销表数据、素材库和运营反馈，转成可发布的朋友圈种草内容。

## 快速运行

直接双击打开：

```text
index.html
```

不需要安装依赖，不需要启动服务，不需要联网。当前版本用本地模拟接口演示完整链路。

## 当前 Agent 链路

```text
飞书营销表同步
-> 产品信息结构化
-> 运营反馈学习
-> 自动找配图
-> 文案生成与去 AI 味
-> 自动排版
-> 发布检查
-> 推荐发布版本
```

## 新增能力

1. **飞书连接配置**：支持输入 App Token、Table ID、View ID；当前用模拟飞书营销表返回数据。
2. **自动找配图**：根据产品类型、关键词、飞书素材表和客户反馈自动选出素材，不再只是给建议。
3. **自动排版**：生成朋友圈单图海报/三图种草卡片/九宫格首图的排版预览。
4. **运营指标调优**：读取曝光、点赞、评论、私聊、成交、热卖品类、潜客沟通摘要。
5. **反馈学习**：根据互动数据和客户问题给出下一轮文案、配图、活动信息优化建议。
6. **接口契约**：左侧链路节点可点击查看输入输出接口，方便面试讲解。

## 面试讲法

这个版本已经从“文案生成器”升级为“私域种草运营 Agent”。它可以接飞书营销表，读取历史互动和素材，自动找配图、生成排版，再根据评论、私聊、成交和潜在客户沟通反馈持续调优。
